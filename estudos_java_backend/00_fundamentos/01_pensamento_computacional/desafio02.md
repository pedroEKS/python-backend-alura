# Desafio: automatizando processos na equipe de RH

## O Cenário

> "Estamos gastando muito tempo com tarefas repetitivas. Será que dá pra facilitar esse processo?"

## Solução: Algoritmo de Automação

### Passo 1: Categorizar os candidatos com a vaga escolhida
```
Para cada candidato:
    Categorizar o candidato pela vaga escolhida
Fim Para
```

### Passo 2: Padronizar as mensagens aos candidatos de acordo com a vaga
```
Para cada candidato:
    Se a vaga for X:
        Enviar mensagem padrão para a vaga X
    Senão Se a vaga for Y:
        Enviar mensagem padrão para a vaga Y
    Fim Se
Fim Para
```

### Passo 3: Padronizar as avaliações dos candidatos
```
Para cada candidato:
    Avaliar o candidato conforme o padrão de avaliação estabelecido
Fim Para
```

### Passo 4: Padronizar os dias e horários de atendimento para turmas de candidatos em comum
```
Para cada grupo de candidatos (mesma vaga):
    Definir dia e horário comum para todos
Fim Para
```

### Passo 5: Definir dias de retorno das mensagens e chamados dos candidatos de acordo com a vaga
```
Para cada candidato:
    Se a vaga for X:
        Definir dia de retorno para a vaga X
    Senão Se a vaga for Y:
        Definir dia de retorno para a vaga Y
    Fim Se
Fim Para
```

### Passo 6: Definir um dia para formalizar a contratação de todos os candidatos
```
Definir um dia fixo para formalizar as contratações de todos os candidatos selecionados
```
